# Проект: Интеллектуальный банковский ассистент NeoStack (RAG-система)

## 1. Обзор проекта
Данный проект представляет собой реализацию RAG-системы (Retrieval-Augmented Generation) для автоматизации ответов на вопросы клиентов и сотрудников банка **NeoStack**. Система позволяет получать точные ответы на основе закрытой базы знаний банка, минимизируя галлюцинации языковых моделей.

---

## 2. Архитектура решения
Процесс работы системы разделен на два ключевых этапа:
1.  **Индексация:** Загрузка документов, разбиение на чанки и создание векторных представлений (эмбеддингов).
2.  **Поиск и Генерация:** Использование гибридного поиска для извлечения наиболее релевантных контекстов и формирование финального ответа.



---

## 3. Стек технологий
* **Язык:** Python 3.10
* **RAG-фреймворк:** LangChain
* **Векторная БД:** ChromaDB
* **Эмбеддинги:** `sentence-transformers/all-mpnet-base-v2`
* **Алгоритмы поиска:** Векторный (Cosine Similarity) + Ключевой (BM25)

---

## 4. Этапы реализации

### Этап 1: Подготовка базы знаний
База знаний состоит из 7 модулей в формате TXT, охватывающих:
* Условия потребительского кредитования («Универсальный»).
* Параметры накопительных счетов («Смарт-Стек»).
* Специальные программы для молодежи («Neo-Генерация»).
* Программы лояльности и реферальные системы.
* Базу знаний FAQ (потеря карт, изменение дат платежа).

### Этап 2: Обработка данных (Preprocessing)
* **Чанкинг:** Текст разбивается на блоки по 600 символов с перекрытием (overlap) 100 символов.
* **Метаданные:** К каждому чанку привязывается имя исходного файла для обеспечения прозрачности источников.

### Этап 3: Гибридный поиск (Hybrid Search)
Для повышения точности реализован кастомный ретривер, комбинирующий:
* **Semantic Search:** Позволяет находить ответы по смыслу, даже если формулировки клиента отличаются от официальных.
* **BM25 Search:** Гарантирует нахождение документов по строгим терминам (например, "2-НДФЛ", "Мир").

---

## 5. Оценка качества (Evaluation)
Качество системы было проверено на контрольной выборке запросов (Ground Truth).

**Результаты тестирования:**
* **Hit Rate@3: 75.0%** (Релевантный документ найден в Топ-3 результатах).
* **MRR (Mean Reciprocal Rank): 0.58** (Средняя позиция правильного ответа — 1.7).



---

## 6. Инструкция по установке и запуску

1.  **Клонировать репозиторий:**
    ```bash
    git clone [https://github.com/ad-tikhonova/Bank-RAG-Assistant.git](https://github.com/ad-tikhonova/Bank-RAG-Assistant.git)
    cd Bank-RAG-Assistant
    ```

2.  **Установить зависимости:**
    ```bash
    pip install langchain langchain-community chromadb sentence-transformers rank_bm25
    ```

3.  **Запустить ассистента:**
    ```bash
    python neostack_rag.py
    ```
